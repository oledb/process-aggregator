# О проекте

Process Agregator - предварительная версия проекта, которая позволит в будущем упростить разработку логики бизнес-процессов на основе использования паттерна CQRS.  

# Структура папок

* packages - здесь расположены библиотеки, которые непосредственно касаются проекта Process Agregator
* demos - примеры микросервесных проектов, использующих Process Agregator
  * common - общий код для всех примеров
